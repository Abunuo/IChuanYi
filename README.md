# IChuanYi
穿衣助手 WebApp（VueJs demo）

> ### 项目简介： 
    穿衣助手的 WebApp 版设计
> ### 应用技术： 
    VueJS、VueX、scss、ES6、Mock、vue-router、vue-resource、Webpack等 

1. 下载压缩包，解压
```
git clone https://github.com/Abunuo/IChuanYi.git
```
2. 安装依赖包，运行项目
```
npm install  //安装依赖包
npm run start   //运行项目
npm run mockJs  //打开假数据服务
```
